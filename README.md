# volunteers
# Веб-приложение для управления волонтёрскими мероприятиями

## О проекте
Приложение реализует все требования Технического задания ТЗКП

## Роли пользователей
- **Волонтёр** — просмотр, запись на смены, личный кабинет.
- **Организатор** — просмотр своих мероприятий.
- **Администратор** — полный CRUD мероприятий и смен.

## Как запустить локально
1. Склонировать репозиторий в `OpenServer/domains/volunteer-app/`
2. Запустить Open Server → Apache + MySQL.
3. Импортировать БД (файл `database/schema.sql`).
4. Настроить `config/config.php` (копия из `config/config.example.php`).
